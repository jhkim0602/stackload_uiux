StackLoad 기능도
본 프로젝튼는 Web 앱 이다.


1) 사용자/계정
	•	회원가입 / 로그인 / 로그아웃
	•	비밀번호 재설정(이메일 인증 등)
	•	사용자 권한 구분(일반 사용자 / 관리자)
	•	내 프로필 조회/수정(닉네임, 소개, 링크 등)
	•	관심 분야/목표 직무/관심 기술스택 설정

⸻

2) 기술스택 허브
	•	기술스택 카테고리 분류(언어/프레임워크/DB/DevOps 등)
	•	기술스택 목록 조회 / 검색 / 필터 / 정렬
	•	기술스택 상세 페이지(개념, 활용처, 관련 기술 연결)
	•	스택 간 연관 정보 표시(예: React ↔ Next ↔ TS 같은 연결)
  •	각 상세 페이지에는 리액트 공식문서 처럼 완벽한 강의 교안을 제공한다.
  •	기술스택에는 프레임워크도 포함하며 실시간 깃 인기 라이브러리도 확인하고 ai가 해당 라이브러리를 설명 해준다.

⸻

3) 가이드·교안 제작(AI)
	•	기술스택 선택 기반 가이드 생성(학습 흐름 제시)
	•	교안 자동 생성 (일단 레이아웃만 잡아라._리액트공식문서처럼)

⸻

4) 커리어 마이징(개인화)
	•	내 기술스택(주특기/관심/학습중) 관리
	•	목표 직무에 따른 추천 스택 제시
	•	학습/활동 이력 기록(저장 기능 중심)
  이때 활동이력은 깃의 흔히 잔디심기 처럼 구성함
	•	개인 맞춤 성장 경로(로드맵) 제안

⸻

5) 대외활동 탭(크롤링 수집)
	•	해커톤 / 공모전 / 대외활동 목록 수집 및 제공
	•	활동 상세(일정, 요구 사항, 참가 링크 등)_[Ai가 크롤링하여 제공]
	•	관심 기술/분야 기반 필터링 및 추천
	•	즐겨찾기/저장(확장 기능)

⸻

6) 채용 공고 탭(크롤링 + 분석 + 추천)
	•	채용 공고 크롤링 수집 및 목록 제공
	•	공고 검색/필터(지역, 경력, 키워드 등)
	•	공고 상세(요약, 주요 요구사항)
	•	공고 요구 기술스택/프레임워크 자동 추출(AI 분석)
	•	사용자 보유 스택과 공고 요구 스택 매칭 점수 산출
	•	관심 기술/주특기 스택 기반 공고 추천(맞춤 매칭 추천)
	•	공고 저장/지원 준비 체크(확장 기능)

⸻

7) 면접 지원(AI)
	•	목표 직무/기술스택 기반 예상 질문 생성
  •	Ai 화상 면접 시뮬레이션을 지원한다.
	•	답변 가이드 및 핵심 키워드 제시
	•	약점 영역 분석 및 보완 학습(추천 스택/교안 연결)
	•	면접 결과 요약 리포트(확장 기능)

⸻

8) AI 화상 면접(실시간 시뮬레이션)
	•	면접 세션 생성(직무/난이도/시간 선택)
	•	실시간 질문 진행(시나리오 기반)
	•	사용자의 답변을 기반으로 후속 질문 생성
	•	피드백 제공(논리성/키워드/전개 흐름 중심)
	•	세션 종료 후 결과 요약 및 개선 포인트 제공
  •	Ai화상 면접 섹션으로는 모두 목표 직무/기술스택 기반뿐만 아니라 우리 StackLoad에는 가상의 기업이 있고 각 기업 마다의 특성이 있다. 실제 대기업 같은 곳에 지원을 해볼수있는 느낌으로 가상의 기업 모집요강 같은 곳에 사용자가 지원하여 ai 화상 면접을 볼 수 있겠금 구성한다.

⸻

9) 커뮤니티(지식 공유)
	•	게시글 작성/조회/수정/삭제
	•	댓글 작성/조회
	•	기술스택 태그 기반 분류/탐색
	•	질문/답변형 게시판 운영(확장 기능)
	•	신고/블라인드 등 운영 기능(확장 기능)

⸻

10) 운영/관리(관리자 기능_관리자 페이지는 추후 구현한다.)
	•	사용자/게시글/댓글 관리
	•	크롤링 데이터 검수/정리(공고/활동)
	•	기술스택 사전 관리(카테고리/태그)
	•	서비스 로그/통계(확장 기능)
